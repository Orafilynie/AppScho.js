## What is "AppScho" ?

[AppScho](https://www.appscho.com/) is an all-in-one mobile app which centralizes everything students need - class schedules, grades, campus news, internship offers - into one intuitive platform. Built for universities and higher education institutions, AppScho strengthens engagement and simplifies communication, making campus life more connected and efficient.

## Get Started

Jump right in by following the [**Installation Guide**](https://appscho.docs.literate.ink/guides/getting-started/installation/) to set things up.

Once in, explore annotated examples in the [**official documentation**](https://appscho.docs.literate.ink/guides/examples/) or check out the [`examples` folder on GitHub](https://github.com/LiterateInk/appscho.js/tree/main/examples). You can also browse the full [**API Reference**](https://js.appscho.docs.literate.ink/) for details.

Still stuck ? [Open an issue](https://github.com/LiterateInk/AppScho.js/issues) or join the conversation on [Discord](https://literate.ink/discord) and ask for help in the appropriate channel.
